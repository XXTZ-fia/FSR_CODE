# Fashion-Product Datasets

该数据集由以下两部分组成：

- details （商品详情）
- reviews  (商品评论)

数据集分别以15个关键词在亚马逊的时尚分栏中进行商品检索，结果如下：(风格类型 + 检索的商品数量)
